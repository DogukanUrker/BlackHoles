## Requirements 📦

#### Backend

- FastAPI
- Uvicorn

#### Frontend

- React
- React-Dom
- Tailwind
- Node

## Installation ⬇️

### Backend

download source code from Github 💾
`git clone https://github.com/DogukanUrker/BlackHolesAPI.git`

go to directory 📁
`cd BlackHolesAPI`

install requirements.txt 🔽
`pip install -r requirements.txt`

it's ready to run 🎉
`python main.py`

### Frontend
download source code from Github 💾
`git clone https://github.com/DogukanUrker/BlackHoles.git`

go to directory 📁
`cd BlackHoles`

install packages 📦
`npm install`

it's ready to run 🎉
`npm run dev`
